# Connectors Skill

Esta skill contém diretrizes e padrões para criação e manutenção de conectores (HTTP/REST/GraphQL) no contexto do agente `n8n-specialist`.

- Propósito: manter conectores seguros, idempotentes e fáceis de operar.
- Conteúdo: checklist, exemplos, padrões de nomeação e tratamento de erros.

Use `USAGE.md` para instruções rápidas.

Scripts de suporte:

- `C:\Users\santo\.agent\scripts\n8n_tools.py` — listar templates, criar placeholders e importar workflows (quando `N8N_URL` e `N8N_API_KEY` estiverem configurados).
- `C:\Users\santo\.agent\scripts\deploy_templates.ps1` — wrapper PowerShell para deploy-all.
