# Workflow: refatoracao-fase

## Uso
- Workflow padrao para qualquer task ligada ao planejamento por fases.

## Passos
1. Identificar fase, task, criterio de aceite e validacao em `docs/planejamento_fases.md`.
2. Ler os arquivos de contexto afetados antes de editar.
3. Listar impacto, arquivos alvo e risco de regressao.
4. Executar a menor mudanca possivel.
5. Validar apenas o que a task exige.
6. Fechar com arquivos alterados, validacao, status atual e proximo passo.
