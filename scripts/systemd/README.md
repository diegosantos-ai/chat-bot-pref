# Systemd Legacy

Este diretório foi mantido apenas como referencia historica de uma operacao antiga baseada em `systemd`.

Estado atual:
- nao e caminho oficial de execucao do projeto
- nao deve ser usado para novo deploy
- pode conter nomes de unidade e premissas de host especificos do ambiente legado

Diretriz do projeto:
- ambiente local: Docker Compose
- automacao de entrega: GitHub Actions
- infraestrutura alvo: AWS com Terraform

Se voce estiver revisando a base para evolucao arquitetural, trate estes arquivos como legado congelado ate a nova operacao ser definida e validada.
